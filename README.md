## Documentación

### BPMN

Los diagramas pueden ser editados en Visual Studio Code con el plugin [bpmn.io](https://marketplace.visualstudio.com/items?itemName=bpmn-io.vs-code-bpmn-io)

Para consultar el diagrama de los procesos del proyecto, por favor dirígete a `docs/bpmn` donde encontrarás los archivos en formato [BPMN](https://www.bpmn.org/), detalles:

1. [campeonato](docs/bpmn/1.%20campeonato.bpmn): Proceso de la creación de un nuevo campeonato
2. [inscripción](docs/bpmn/2.%20inscripción.bpmn): Proceso de inscripción de un nuevo equipo para el campeonato
3. [calendario](docs/bpmn/3.%20calendario.bpmn): Proceso de organización de modalidad del campeonato y definición fechas
4. [partidos](docs/bpmn/4.%20partidos.bpmn): Proceso de gestión de los datos del partido y posterior validación.
5. [resultados](docs/bpmn/5.%20resultados.bpmn): Proceso de validar los resultados de la fecha.
6. [notificaciones](docs/bpmn/6.%20notificaciones.bpmn): Proceso de validar los resultados de la fecha.
7. [finalización](docs/bpmn/7.%20finalización.bpmn): Proceso de cierre de un campeonato.
